December 2014

I started looking at my remote location and determined that I need a solar power station to at least half power the server. The rest will be done by on-site AC power. A friend gave me a computer tower to serve as my server. I intend to swap out the hard drive and install Ubuntu server to run everything. A USB Wi-Fi dongle will be plugged into the server for communications. That way I can simply SSH into the server to make any and all changes. A small wired network will need to be set up on-site and I'm thinking I might need power over Ethernet to run the sensors.
